# 1945 Shooter Game
1945 Shooter Game is an arcade-style shooter in which the player controls an aircraft, shooting enemies and trying to survive as long as possible. Developed in C using the raylib library.
## User Guide

| Key | Action |
| :---: | :---: |
| **Up Arrow** | Move Up |
| **Down Arrow** | Move Down |
| **Right Arrow** | Move Right |
| **Left Arrow** | Move Left |
| **Space** | Shoot |

Lives and Score:

The player starts with 3 lives.

Each destroyed enemy increases the score.

The game ends when all lives are lost.

## Developer Guidelines
Requirements to Compile:

C compiler (GCC, Clang, MSVC)

raylib installed https://www.raylib.com/

Required audio and graphics libraries for raylib
